# Phase 2: llm-harness-loop Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the `llm-harness-loop` crate — a stateless agent loop that drives LLM streaming, executes tool batches, and emits `AgentEvent` streams.

**Architecture:** `agent_loop()` returns a cold `impl Stream<Item = AgentEvent>` powered by `async_stream::stream!`. All state is local to the function body. The `llm_adapter::provider::Provider` trait is the LLM client interface; a thin `MockLlmClient` (behind `test-utils` feature) enables deterministic testing without network calls.

**Tech Stack:** Rust 2024, `llm-harness-types` (workspace), `llm_adapter` (path dep), `async-stream 0.3`, `tokio 1` (full), `futures 0.3`, `uuid 1`, `chrono 0.4`

## Adapter Interface Reality Check

The actual `llm_adapter` crate differs from the spec's assumed interface. Key mappings:

| Spec assumed | Actual `llm_adapter` |
|---|---|
| `LlmClient::stream(model, messages, system, tools, opts, auth)` | `Provider::chat_stream(&ChatRequest) -> Result<StreamHandle, LlmError>` |
| `StreamEvent::MessageStart { id, model, provider, api }` | No equivalent — generate UUID at stream start |
| `StreamEvent::TextDelta { text }` | `StreamEvent::TextDelta { index, text }` |
| `StreamEvent::ThinkingDelta { thinking, signature }` | `StreamEvent::ReasoningDelta { index, text }` (no signature in stream) |
| `StreamEvent::ToolUseStart { id, name }` | `StreamEvent::ContentStart { index, kind: ToolInvocation { id, name } }` |
| `StreamEvent::ToolUseDelta { id, partial_input }` | `StreamEvent::ToolDelta { index, arguments }` |
| `StreamEvent::ToolUseEnd { id, input }` | `StreamEvent::ContentStop { index }` (parse JSON from accumulated deltas) |
| `StreamEvent::MessageEnd { stop_reason, usage }` | `StreamEvent::MessageStop { stop_reason, usage }` |
| `StreamEvent::Error(e)` | `Result::Err(LlmError)` stream item |

`StreamHandle::new()` is `pub(crate)` — Task 1 adds `pub fn from_raw_stream()` to enable `MockLlmClient`.

## Spec Gaps Filled

The spec's `LoopConfig` omits fields needed by the actual implementation. These additions are required:

| Added field | Type | Why |
|---|---|---|
| `model` | `String` | `ChatRequest` needs a model string |
| `thinking_level` | `ThinkingLevel` | stored for future ThinkingLevel→adapter mapping |
| `max_tokens` | `u32` | `ChatRequest` requires `max_tokens` |
| `temperature` | `Option<f32>` | passed to `ChatRequest` |
| `env` | `Arc<dyn ExecutionEnv>` | `ToolContext` requires it |
| `abort` | `CancellationToken` | `ToolContext` requires it; loop checks it |

`AuthHook`, `StreamOptions.headers/timeout_ms/max_retries` are kept in `LoopConfig` but not forwarded to the current adapter (adapter does not support per-request auth/headers). They're stored for future extensibility.

```rust
pub(crate) fn convert_stop_reason(r: AdapterStop) -> StopReason {
    match r {
        AdapterStop::EndTurn => StopReason::EndTurn,
        AdapterStop::MaxTokens => StopReason::MaxTokens,
        AdapterStop::StopSequence => StopReason::StopSequence,
        AdapterStop::ToolUse => StopReason::ToolUse,
        AdapterStop::ContentFilter | AdapterStop::Other(_) => StopReason::Other,
    }
}

pub(crate) fn convert_usage(u: Usage) -> TokenUsage {
    TokenUsage {
        input_tokens: u.input_tokens,
        output_tokens: u.output_tokens,
        cache_read_tokens: u.cached_input_tokens,
        cache_creation_tokens: u.cache_creation_input_tokens,
    }
}

pub(crate) fn tools_to_defs(tools: &[Arc<dyn Tool>]) -> Vec<ToolDef> {
    tools.iter().map(|t| ToolDef::new(
        t.name(),
        t.description(),
        t.parameters_schema().clone(),
    )).collect()
}

pub(crate) fn content_block_to_request(cb: &ContentBlock) -> Option<RequestContent> {
    match cb {
        ContentBlock::Text { text } => Some(RequestContent::Text(text.clone())),
        ContentBlock::Image { source } => {
            let src = match source {
                ImageSource::Base64 { media_type, data } =>
                    llm_adapter::types::ImageSource::Base64 {
                        media_type: media_type.clone(),
                        data: data.clone(),
                    },
            };
            Some(RequestContent::Image(src))
        }
        ContentBlock::Thinking { .. } | ContentBlock::ToolUse { .. } => None,
    }
}

pub(crate) fn content_block_to_response(cb: &ContentBlock) -> Option<ResponseContent> {
    match cb {
        ContentBlock::Text { text } => Some(ResponseContent::Text(text.clone())),
        ContentBlock::Thinking { thinking, signature } => Some(ResponseContent::Reasoning {
            text: thinking.clone(),
            signature: signature.clone(),
        }),
        ContentBlock::ToolUse { id, name, input } => Some(ResponseContent::ToolInvocation(
            ToolInvocation { id: id.clone(), name: name.clone(), input: input.clone() },
        )),
        ContentBlock::Image { .. } => None,
    }
}
```

```rust
impl ConvertToLlmHook for DefaultConvertToLlm {
    fn convert<'a>(
        &'a self,
        messages: &'a [AgentMessage],
    ) -> BoxFuture<'a, Result<Vec<AdapterMessage>, AgentError>> {
        Box::pin(async move {
            use crate::type_bridge::{content_block_to_request, content_block_to_response};
            let mut out = Vec::with_capacity(messages.len());
            for msg in messages {
                match msg {
                    AgentMessage::User(u) => {
                        let content = u.content.iter().filter_map(content_block_to_request).collect();
                        out.push(AdapterMessage::User(content));
                    }
                    AgentMessage::Assistant(a) => {
                        let content = a.content.iter().filter_map(content_block_to_response).collect();
                        out.push(AdapterMessage::Assistant(content));
                    }
                    AgentMessage::ToolResult(t) => {
                        let content = t.content.iter().filter_map(content_block_to_request).collect();
                        out.push(AdapterMessage::Tool {
                            invocation_id: t.tool_use_id.clone(),
                            content,
                            is_error: t.is_error,
                        });
                    }
                    AgentMessage::BranchSummary(b) => {
                        let text = format!("<summary>\n{}\n</summary>", b.summary);
                        out.push(AdapterMessage::User(vec![RequestContent::Text(text)]));
                    }
                    AgentMessage::CompactionSummary(c) => {
                        let text = format!("<summary>\n{}\n</summary>", c.summary);
                        out.push(AdapterMessage::User(vec![RequestContent::Text(text)]));
                    }
                    AgentMessage::Custom(c) => {
                        match &self.custom_handler {
                            Some(h) => out.push(h.convert(c).await?),
                            None => return Err(AgentError::Internal(
                                "custom message has no converter; provide CustomMessageConverter via DefaultConvertToLlm::with_custom_converter".into()
                            )),
                        }
                    }
                }
            }
            Ok(out)
        })
    }
}
```

```rust
pub(crate) fn group_by_sequential(
    tools: &[Arc<dyn Tool>],
    default_mode: ToolExecutionMode,
) -> Vec<Vec<usize>> {
    if tools.is_empty() { return vec![]; }
    let mut groups: Vec<Vec<usize>> = vec![vec![]];
    for (i, t) in tools.iter().enumerate() {
        let mode = t.execution_mode();
        let effective = if mode == ToolExecutionMode::Sequential {
            ToolExecutionMode::Sequential
        } else {
            default_mode
        };
        if effective == ToolExecutionMode::Sequential {
            // Close current group if non-empty, put this tool alone, start new group
            if !groups.last().unwrap().is_empty() {
                groups.push(vec![]);
            }
            groups.last_mut().unwrap().push(i);
            groups.push(vec![]);
        } else {
            groups.last_mut().unwrap().push(i);
        }
    }
    // Remove trailing empty group
    groups.retain(|g| !g.is_empty());
    groups
}

pub(crate) async fn execute_tool_batch(
    calls: Vec<(String, serde_json::Value, Arc<dyn Tool>)>,
    ctx_factory: impl Fn(String) -> ToolContext,
    default_mode: ToolExecutionMode,
) -> Vec<(String, Result<ToolResult, ToolError>)> {
    let tools: Vec<Arc<dyn Tool>> = calls.iter().map(|(_, _, t)| t.clone()).collect();
    let groups = group_by_sequential(&tools, default_mode);

    // Pre-build per-call results vector with placeholders
    let n = calls.len();
    let mut results: Vec<Option<(String, Result<ToolResult, ToolError>)>> = (0..n).map(|_| None).collect();

    let calls = Arc::new(calls);

    for group in groups {
        let futures: Vec<_> = group.iter().map(|&idx| {
            let (tool_use_id, args, tool) = &calls[idx];
            let ctx = ctx_factory(tool_use_id.clone());
            let tool = tool.clone();
            let args = args.clone();
            let tool_use_id = tool_use_id.clone();
            async move {
                let r = tool.execute(args, &ctx).await;
                (idx, tool_use_id, r)
            }
        }).collect();

        let group_results = join_all(futures).await;
        for (idx, id, r) in group_results {
            results[idx] = Some((id, r));
        }
    }

    results.into_iter().map(|r| r.unwrap()).collect()
}
```

```rust
impl StreamingState {
    pub(crate) fn process(&mut self, event: AdapterEvent) -> Vec<AgentEvent> {
        use crate::type_bridge::{convert_stop_reason, convert_usage};
        let mid = self.message_id.clone();

        match event {
            AdapterEvent::ContentStart { index, kind } => {
                match kind {
                    ContentKind::Text => {
                        self.text_blocks.insert(index, String::new());
                        self.block_order.push((index, BlockKind::Text));
                        vec![]
                    }
                    ContentKind::Reasoning => {
                        self.reasoning_blocks.insert(index, String::new());
                        self.block_order.push((index, BlockKind::Reasoning));
                        vec![]
                    }
                    ContentKind::ToolInvocation { id, name } => {
                        self.tool_blocks.insert(index, (id.clone(), name.clone(), String::new()));
                        self.block_order.push((index, BlockKind::Tool));
                        vec![AgentEvent::ToolCallStart { message_id: mid, tool_use_id: id, name }]
                    }
                }
            }
            AdapterEvent::TextDelta { index, text } => {
                self.text_blocks.entry(index).or_default().push_str(&text);
                let partial = self.partial_message();
                vec![
                    AgentEvent::TextDelta { message_id: mid.clone(), text },
                    AgentEvent::MessageUpdate { message_id: mid, partial },
                ]
            }
            AdapterEvent::ReasoningDelta { index, text } => {
                self.reasoning_blocks.entry(index).or_default().push_str(&text);
                let partial = self.partial_message();
                vec![
                    AgentEvent::ThinkingDelta { message_id: mid.clone(), thinking: text, signature: None },
                    AgentEvent::MessageUpdate { message_id: mid, partial },
                ]
            }
            AdapterEvent::ToolDelta { index, arguments } => {
                if let Some((id, _, args)) = self.tool_blocks.get_mut(&index) {
                    args.push_str(&arguments);
                    let id = id.clone();
                    vec![AgentEvent::ToolCallArgsDelta { tool_use_id: id, partial_input: arguments }]
                } else { vec![] }
            }
            AdapterEvent::ContentStop { index } => {
                // Only tool blocks produce an event at ContentStop
                if let Some((id, _, args)) = self.tool_blocks.get(&index) {
                    let parsed = serde_json::from_str(args).unwrap_or_else(|_| serde_json::Value::Null);
                    vec![AgentEvent::ToolCallEnd { tool_use_id: id.clone(), args: parsed }]
                } else { vec![] }
            }
            AdapterEvent::MessageStop { stop_reason, usage } => {
                let final_content = self.build_final_content();
                let message = AssistantMessage {
                    content: final_content,
                    stop_reason: Some(convert_stop_reason(stop_reason)),
                    timestamp: Utc::now(),
                    provider: None,
                    api: None,
                    model: Some(self.model.clone()),
                    usage: Some(convert_usage(usage)),
                    error_message: None,
                };
                vec![AgentEvent::MessageEnd { message_id: mid, message }]
            }
        }
    }

    pub(crate) fn partial_message(&self) -> AssistantMessage {
        AssistantMessage {
            content: self.build_final_content(),
            stop_reason: None,
            timestamp: Utc::now(),
            provider: None,
            api: None,
            model: Some(self.model.clone()),
            usage: None,
            error_message: None,
        }
    }

    fn build_final_content(&self) -> Vec<ContentBlock> {
        let mut indexed: Vec<(usize, ContentBlock)> = Vec::new();
        for (idx, kind) in &self.block_order {
            let cb = match kind {
                BlockKind::Text => {
                    let text = self.text_blocks.get(idx).cloned().unwrap_or_default();
                    ContentBlock::Text { text }
                }
                BlockKind::Reasoning => {
                    let thinking = self.reasoning_blocks.get(idx).cloned().unwrap_or_default();
                    ContentBlock::Thinking { thinking, signature: None }
                }
                BlockKind::Tool => {
                    if let Some((id, name, args)) = self.tool_blocks.get(idx) {
                        let input = serde_json::from_str(args).unwrap_or(serde_json::Value::Null);
                        ContentBlock::ToolUse { id: id.clone(), name: name.clone(), input }
                    } else { continue; }
                }
            };
            // Deduplicate by index
            if !indexed.iter().any(|(i, _)| i == idx) {
                indexed.push((*idx, cb));
            }
        }
        indexed.sort_by_key(|(i, _)| *i);
        indexed.into_iter().map(|(_, cb)| cb).collect()
    }
}
```

```rust
fn run_loop(
    client: Arc<dyn Provider>,
    mut ctx: AgentContext,
    mut config: LoopConfig,
    drain_steer_first: bool,
) -> impl Stream<Item = AgentEvent> + Send {
    async_stream::stream! {
        use crate::{
            dispatch::execute_tool_batch,
            stream_state::StreamingState,
            type_bridge::tools_to_defs,
        };
        use llm_adapter::types::{ChatRequest, ToolChoice};
        use futures::StreamExt;
        use tokio::sync::mpsc;

        // Emit AgentStart. For agent_loop_continue, initial_messages is empty.
        let initial_messages: Vec<AgentMessage> = vec![];
        yield AgentEvent::AgentStart { initial_messages: initial_messages.clone() };
        ctx.messages.extend(initial_messages);

        // Drain stale steer messages for agent_loop_continue
        if drain_steer_first {
            if let Some(ref mut rx) = config.steer_rx {
                while let Ok(_) = rx.try_recv() {}
            }
        }

        let snapshot_start = ctx.messages.len();
        let mut turn_index = 0u32;
        let mut new_messages: Vec<AgentMessage> = vec![];

        'main: loop {
            // Check abort
            if config.abort.is_cancelled() {
                yield AgentEvent::Error(AgentError::Aborted);
                break 'main;
            }

            // Drain steer messages (between turns, not on first turn for normal loop)
            if turn_index > 0 || drain_steer_first {
                if let Some(ref mut rx) = config.steer_rx {
                    while let Ok(msg) = rx.try_recv() {
                        ctx.messages.push(msg.clone());
                        new_messages.push(msg);
                    }
                }
            }

            // TransformContextHook
            if let Some(h) = &config.transform_context {
                match h.transform(ctx.clone()).await {
                    Ok(new_ctx) => ctx = new_ctx,
                    Err(e) => {
                        yield AgentEvent::Error(e);
                        break 'main;
                    }
                }
            }

            yield AgentEvent::TurnStart { index: turn_index };

            // Build ChatRequest
            let llm_messages = match config.convert_to_llm.convert(&ctx.messages).await {
                Ok(msgs) => msgs,
                Err(e) => {
                    yield AgentEvent::Error(e);
                    break 'main;
                }
            };
            let tool_defs = tools_to_defs(&config.tools);
            let tool_choice = if tool_defs.is_empty() { ToolChoice::None } else { ToolChoice::Auto };

            let mut req_builder = ChatRequest::builder(&config.model, config.max_tokens)
                .messages(llm_messages)
                .tools(tool_defs)
                .tool_choice(tool_choice);
            if let Some(t) = config.temperature { req_builder = req_builder.temperature(t); }
            if let Some(sp) = &ctx.system_prompt {
                // Prepend system message by rebuilding with system as first message
                use llm_adapter::types::Message as AMsg;
                let mut msgs_with_system = vec![AMsg::System(sp.clone())];
                msgs_with_system.extend(match config.convert_to_llm.convert(&ctx.messages).await {
                    Ok(m) => m,
                    Err(e) => { yield AgentEvent::Error(e); break 'main; }
                });
                req_builder = ChatRequest::builder(&config.model, config.max_tokens)
                    .messages(msgs_with_system)
                    .tools(tools_to_defs(&config.tools))
                    .tool_choice(if config.tools.is_empty() { ToolChoice::None } else { ToolChoice::Auto });
                if let Some(t) = config.temperature { req_builder = req_builder.temperature(t); }
            }
            let req = req_builder.build();

            // Call LLM
            let mut handle = match client.chat_stream(&req).await {
                Ok(h) => h,
                Err(e) => {
                    yield AgentEvent::Error(AgentError::Provider(e.to_string()));
                    break 'main;
                }
            };

            // Emit MessageStart
            let mut state = StreamingState::new(handle.model.clone());
            let message_id = state.message_id.clone();
            yield AgentEvent::MessageStart { message_id: message_id.clone() };

            // Process stream events
            let mut final_message: Option<AssistantMessage> = None;
            let stream = handle.events();
            while let Some(item) = stream.next().await {
                match item {
                    Ok(adapter_event) => {
                        for agent_event in state.process(adapter_event) {
                            if let AgentEvent::MessageEnd { ref message, .. } = agent_event {
                                final_message = Some(message.clone());
                            }
                            yield agent_event;
                        }
                    }
                    Err(e) => {
                        yield AgentEvent::Error(AgentError::Provider(e.to_string()));
                        break 'main;
                    }
                }
            }

            let assistant_msg = match final_message {
                Some(m) => m,
                None => {
                    yield AgentEvent::Error(AgentError::Internal("stream ended without MessageEnd".into()));
                    break 'main;
                }
            };

            // Add assistant message to context and new_messages
            let assistant_agent_msg = AgentMessage::Assistant(assistant_msg.clone());
            ctx.messages.push(assistant_agent_msg.clone());
            new_messages.push(assistant_agent_msg);

            let stop_reason = assistant_msg.stop_reason.unwrap_or(StopReason::EndTurn);

            // Tool execution
            let mut tool_results: Vec<(String, Result<ToolResult, ToolError>)> = vec![];
            if stop_reason == StopReason::ToolUse {
                // Collect tool use blocks
                let tool_calls: Vec<(String, String, serde_json::Value)> = assistant_msg.content.iter()
                    .filter_map(|cb| if let ContentBlock::ToolUse { id, name, input } = cb {
                        Some((id.clone(), name.clone(), input.clone()))
                    } else { None })
                    .collect();

                // Build calls with matching tool implementations
                let calls: Vec<(String, serde_json::Value, Arc<dyn Tool>)> = tool_calls.iter()
                    .filter_map(|(id, name, args)| {
                        config.tools.iter().find(|t| t.name() == name)
                            .map(|t| (id.clone(), args.clone(), t.clone()))
                    })
                    .collect();

                let assistant_arc = Arc::new(assistant_msg.clone());
                let env = config.env.clone();
                let abort = config.abort.clone();
                let turn_idx = turn_index;

                // Emit ToolExecutionStart for each call
                for (id, name, args) in &tool_calls {
                    yield AgentEvent::ToolExecutionStart {
                        tool_use_id: id.clone(),
                        tool_name: name.clone(),
                        args: args.clone(),
                    };
                }

                // Execute tool batch
                let results = execute_tool_batch(
                    calls,
                    {
                        let env = env.clone();
                        let abort = abort.clone();
                        let assistant_arc = assistant_arc.clone();
                        move |tool_use_id| {
                            let (tx, _rx) = mpsc::channel(16);
                            ToolContext {
                                env: env.clone(),
                                abort: abort.clone(),
                                tool_use_id,
                                turn_index: turn_idx,
                                assistant_message: assistant_arc.clone(),
                                update_tx: tx,
                            }
                        }
                    },
                    config.default_execution_mode,
                ).await;

                // Emit ToolExecutionEnd and build ToolResult messages
                for (id, result) in &results {
                    yield AgentEvent::ToolExecutionEnd {
                        tool_use_id: id.clone(),
                        result: result.as_ref().map(|r| r.clone()).map_err(|e| ToolError::Execution(e.to_string())),
                    };
                    let content = match result {
                        Ok(r) => r.content.clone(),
                        Err(e) => vec![ContentBlock::Text { text: e.to_string() }],
                    };
                    let tool_result_msg = AgentMessage::ToolResult(ToolResultMessage {
                        tool_use_id: id.clone(),
                        content,
                        is_error: result.is_err(),
                        timestamp: chrono::Utc::now(),
                    });
                    ctx.messages.push(tool_result_msg.clone());
                    new_messages.push(tool_result_msg);
                }
                tool_results = results;

                // TurnEnd
                yield AgentEvent::TurnEnd {
                    index: turn_index,
                    message: assistant_msg.clone(),
                    tool_results: tool_results.clone(),
                };

                // Check terminate condition
                let all_terminate = tool_results.iter().all(|(_, r)| {
                    r.as_ref().map(|tr| tr.terminate).unwrap_or(false)
                });
                if all_terminate && !tool_results.is_empty() {
                    break 'main;
                }
            } else {
                // Non-tool stop
                yield AgentEvent::TurnEnd {
                    index: turn_index,
                    message: assistant_msg.clone(),
                    tool_results: vec![],
                };

                // ShouldStopHook
                let should_stop = if let Some(h) = &config.should_stop {
                    h.should_stop(ShouldStopCtx {
                        last_assistant: &assistant_msg,
                        stop_reason,
                        turn_index,
                    }).await
                } else {
                    true // default: stop
                };

                if should_stop {
                    // Check follow_up channel
                    if let Some(ref mut rx) = config.follow_up_rx {
                        if let Ok(follow_up_msg) = rx.try_recv() {
                            ctx.messages.push(follow_up_msg.clone());
                            new_messages.push(follow_up_msg);
                            turn_index += 1;
                            continue 'main;
                        }
                    }
                    break 'main;
                }
                // should_stop = false → continue with PrepareNextTurn
            }

            // PrepareNextTurnHook (called when continuing)
            if let Some(h) = &config.prepare_next_turn {
                let last_tool_results = &tool_results;
                match h.prepare(PrepareNextTurnCtx {
                    turn_index,
                    last_message: &assistant_msg,
                    last_tool_results,
                }).await {
                    Ok(directive) => {
                        if let Some(new_ctx) = directive.context { ctx = new_ctx; }
                        if let Some(m) = directive.model { config.model = m; }
                        if let Some(tl) = directive.thinking_level { config.thinking_level = tl; }
                        if let Some(tools) = directive.tools { config.tools = tools; }
                    }
                    Err(e) => {
                        yield AgentEvent::Error(e);
                        break 'main;
                    }
                }
            }

            turn_index += 1;
        }

        yield AgentEvent::AgentEnd { new_messages };
    }
}
```
