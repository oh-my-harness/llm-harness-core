# llm-harness-core SDK Guides

These guides explain how to use `llm-harness-core` as an agent framework SDK.
They are user-facing companions to the lower-level design specs under
`docs/superpowers/specs/`.

- [Quick Start](quick-start.md): use `Agent` for lightweight scripts and tests.
- [AgentHarness](agent-harness.md): build session-backed agents.
- [Tool Authoring](tool-authoring.md): implement custom tools.
- [Session Model](session-model.md): understand entries, branches, and context.
- [Hooks](hooks.md): customize behavior around turns, tools, and compaction.
- [Core / Runtime Boundary](core-runtime-boundary.md): decide what belongs in
  core, runtime, or a domain agent.
